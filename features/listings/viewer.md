# Viewer

The media viewer opens media in fullscreen directly within a listing, without navigating away from the page. Available keyboard shortcuts are shown at the bottom of the viewer.

## Opening the viewer

Click on any media thumbnail in a listing to open it in the viewer.

## Navigation

| Action          | Key / Control                |
| --------------- | ---------------------------- |
| Next media      | `→` (Right arrow)            |
| Previous media  | `←` (Left arrow)             |
| Close viewer    | `Esc`                        |

## Actions in the viewer

While a media item is open in the viewer, the same [Actions](actions.md) shortcuts are available (`L` to like, `S` to share, `E` to edit, `Del` to delete, etc.).

To open the full media page, click the title link displayed at the top of the viewer.
