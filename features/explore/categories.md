# Categories

Categories are labels used to organize content across the site. Any image can be assigned to a category, and each category gets its own public listing page.

## Browsing a category

Category pages are accessible at:

```plain
/category/{url-key}
```

For example, a category with URL key `art` is browsable at `/category/art`.

Each category page displays:

* The category **name** and optional **description** at the top
* A content listing sortable by: **Most recent**, **Oldest**, **Most viewed**, **Most liked**, **AZ**
* The standard **All / Actions** bulk action menu
